
# RoutePlanner – Frontend

RoutePlanner is a web application for planning hiking and biking routes.  
The frontend provides a clean, modern UI for creating routes, viewing interactive maps, and managing saved trips.

---

## Features

-  **Authenticated experience** – personalized dashboard for logged-in users  
-  **Create New Route**  
  - Choose destination (city / area)  
  - Select trip type: **Hike** or **Bike**  
  - Choose loop / city-to-city style  
-  **Interactive Map** – view the generated route on a map  
- **Route details** – distance, daily breakdown and short description  
-  **Saved Routes & History** – manage previously generated routes  
-  **Login / Logout flow** with protected pages

*(You can add/remove bullets according to what you finally implemented.)*

---

##  Tech Stack

- **React**
- **React Router** – client-side routing
- **State management** – (e.g. Zustand / Context API)
- **Map library** – Leaflet / `react-leaflet`
- **Fetch / Axios** – HTTP requests to the backend API
- CSS / custom styling
